## "Thinking Of You" Email Project

#### Running this project
To open this project, please download all individual items in the "Email Project" directory just as they are found. assets/ should all remain in the same folder, which should be at the same level as "styles.css" and "index.html". Then, opening the index.html file should provide full access to all included media.

#### Media limitations
The GIFs contained in this project should run in most email clients. If they do not render in your email client, please consider using a different one or just read the alt text. Video support may be more limited, so it is suggested to open this email in a browser or to benefit from the alt text included with the fallback images.
